# 🚚 App Delivery - Backend (Xano)

Este repositório contém a estrutura do backend do projeto App Delivery, desenvolvido utilizando o Xano.

## 🧠 Tecnologias

- Xano (Backend no-code)
- API REST
- Banco de dados relacional

---

## 🗄️ Estrutura do Banco de Dados

A estrutura completa do banco pode ser visualizada aqui:

👉 [Ver banco de dados](./database.md)

---

## 🔗 Funcionalidades da API

O backend é responsável por:

- 👤 Cadastro e autenticação de usuários
- 📦 Gerenciamento de pedidos
- 🛒 Controle de produtos
- 📍 Endereços de entrega
- 🔄 Status de pedidos

---

## 📱 Integração

Este backend é utilizado pelo frontend desenvolvido em FlutterFlow.

---

## ⚠️ Observações

- Dados sensíveis não estão incluídos
- Este repositório é para fins de documentação e portfólio
